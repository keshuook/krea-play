# krea-play

## Features

- **Login Page:** Allow login only through krea affiliated emails.
- **Home Page:** Allows navigation to different categories (badminton, football, basketball, volleyball, tennis).
- **In Category:** Show various timeslots and colour coordinate the slots based on how full the court is.
- **Admin Page:** Book field in advance
